# Kişisel Tanıtım Web Sitesi

Bu proje, HTML, CSS, JavaScript ve PHP kullanılarak geliştirilmiş çok sayfalı bir kişisel tanıtım web sitesidir.  
Proje, temel web teknolojilerini ve PHP ile form işlemlerini göstermek amacıyla hazırlanmıştır.

## Kullanılan Teknolojiler
- HTML5  
- CSS3  
- JavaScript  
- PHP  

## Proje İçeriği
- Ana sayfa, hakkımda, özgeçmiş, şehir tanıtımı ve iletişim sayfaları  
- JavaScript ile temel kullanıcı etkileşimleri  
- PHP kullanılarak:
  - Kullanıcı giriş işlemleri  
  - İletişim formu verilerinin işlenmesi  

## Amaç
Bu proje, temel web geliştirme bilgilerini uygulamalı olarak göstermek ve  
PHP ile form tabanlı işlemler konusunda pratik yapmak amacıyla geliştirilmiştir.

## Kurulum
1. Proje dosyalarını bilgisayarınıza indirin
2. Bir PHP destekli sunucu ortamında çalıştırın  
   - XAMPP, WAMP veya benzeri
3. Proje klasörünü `htdocs` (veya eşdeğeri) dizinine taşıyın
4. Tarayıcıdan proje dizinine gidin

## Notlar
- Proje eğitim ve öğrenme amaçlıdır
- Framework kullanılmamıştır
- Basit ve anlaşılır bir yapı hedeflenmiştir
